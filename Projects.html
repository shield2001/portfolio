DE Project 2: The second project that I took part in was the Entrepreneurship Project. The goal of this project was to create a solution to a real world problem using the specific skill sets that we have acquired in the Digital Electronics class.
The problem that my group decided to fix, was the issue of drivers getting stressed at red lights, also we came up for a solution to reduce environmental pollution. Our product that would solve these problems was called "Redvolution". What my group decided to do was make a count down timer that would be installed on the red stoplight.
The count down timer would visually display the remaining amount of time that the driver has to wait at the red light. This will reduce the stress that the drivers experience because they'll know how long they'll have to wait they won't have to constantly worry about it. This in turn will reduce pollution because if there is a particularly long wait,
then the driver could just turn off their car and wait for the light to turn. When the car is off it can't pollute the environment, so the product will reduce environmental pollution in that way. My responsibility for the project was to create the PowerPoint presentation to use to present our product to the rest of the class.

CSE Project 1: This project was the make a game project in Scratch. The aim of the project was to use the tools available in Scratch to make a functioning game or something like a game.
My part was to make the game in Scratch while my partner's aim was to do the documentation for the project overall. During its creation, the game had some challenges. The first of which was making an arcade style game using the limited functions available in Scratch.
A second problem was figuring out how to make the main character throw the ball. We ran into issues when the input to throw was given after a ball had already been thrown, but before the ball had reached the opponent, the game would reset the ball back to the main character and the ball would never reach the other person.
